## 📌 About The Project

Lechelle is a modern restaurant web experience designed to reflect the elegance of contemporary continental and Italian cuisine. The website focuses on delivering a premium digital dining experience through clean layouts, refined visuals, responsive design, and smooth user interactions.

Built with a modern frontend architecture, the project emphasizes performance, responsiveness, scalability, and production-ready deployment practices.

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/masterwayne22/lechelle.git
```

### Navigate Into Project

```bash
cd lechelle
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```
